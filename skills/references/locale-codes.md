# Locale Code Reference

Quick reference for common BCP-47 locale codes used in OpenMeme localization.

## Spanish Variants

| Code | Region | Notes |
|------|--------|-------|
| es-AR | Argentina | Voseo, lunfardo, specific local slang |
| es-MX | Mexico | Mexican Spanish conventions |
| es-ES | Spain | Castilian, vosotros form |
| es-CO | Colombia | Colombian conventions |
| es-CL | Chile | Chilean Spanish, unique slang |
| es-PE | Peru | Peruvian conventions |
| es-UY | Uruguay | Close to Argentine, subtle differences |

## Portuguese Variants

| Code | Region | Notes |
|------|--------|-------|
| pt-BR | Brazil | Brazilian Portuguese |
| pt-PT | Portugal | European Portuguese |
