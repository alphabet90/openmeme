# SEO Standards for OpenMeme

Reference guide for optimizing meme metadata for searchability and discoverability.

## Title SEO

### Length
- **Ideal**: 30-70 characters
- **Minimum**: 20 characters
- **Maximum**: 80 characters (hard cap)

### Structure
Use this priority order when composing:
```
[Character Name] [Action/Scene] — [Format/Context]
```

Examples:
- `Homer Strangling Bart Animated Reaction Template` (50 chars)
- `Messi Lifting World Cup — Argentina Celebration Meme` (53 chars)
- `Drake Hotline Bling Rejection Format` (38 chars)

### Anti-patterns
| Bad | Why | Fix |
|-----|-----|-----|
| `GOLAZO!!!!` | ALL CAPS, no context | `Messi Iconic Goal Celebration` |
| `simpson` | Too short, generic | `Homer Simpson Bush GIF Reaction` |
| `homer homer homer` | Keyword stuffing | `Homer Simpson Brain Exit Meme` |
| `🔥HOMER🔥STRANGLING🔥` | Emoji, symbols | `Homer Strangling Bart Scene` |
| `the simpsons season 5 episode 3 scene where homer...` | Too long | `Homer Workplace Frustration Reaction` |

### Capitalization
- Sentence case for English titles
- Keep proper nouns capitalized
- Do not capitalize every word (Title Case Only For Names)

## Description SEO

### Length
- **Ideal**: 80-200 characters
- **Minimum**: 60 characters
- **Maximum**: 250 characters (hard cap)

### Structure
```
[Visual description: who, what, where]. [Usage context: when/how used]. [Origin note: franchise/era if relevant].
```

### Template Examples

**Character reaction**:
```
{Character} from {Show} with {expression} face, {action}. Popular reaction image for {emotion} moments in {context}. From {Show} Season {N}.
```

**Template/format**:
```
{Format name} showing {setup} vs {punchline}. Two-panel exploitable template used for {use cases}. Originates from {Source}.
```

**Surreal/absurdist**:
```
Surreal image of {description}. Absurdist humor meme used in {context}. Popular in {platform/region} shitposting communities.
```

### Keywords to Include
When relevant, work in these high-search terms naturally:
- Character name + franchise (`Homer Simpson`, `SpongeBob`)
- Format type (`reaction image`, `template`, `macro`, ` exploitable`)
- Emotion (`angry`, `happy`, `sad`, `confused`, `surprised`)
- Specific action (`strangling`, `crying`, `pointing`, `dancing`)
- Language/region (`spanish`, `argentinian`, `latin-american`)

## Tag SEO

### Minimum Viable Tag Set (5-12 tags)

Every meme should have tags from at least these categories:

| Category | Examples | Required? |
|----------|----------|-----------|
| **Franchise** | `simpsons`, `spongebob`, `star-wars` | Yes, if applicable |
| **Character** | `homer-simpson`, `patrick-star` | Yes, if applicable |
| **Emotion** | `angry`, `happy`, `sad`, `confused` | Yes |
| **Format** | `reaction`, `template`, `macro`, `shitpost` | Yes |
| **Region/Language** | `argentina`, `spanish`, `latam` | Yes, if applicable |
| **Specific Action** | `strangling`, `crying`, `pointing` | Recommended |
| **Category Slug** | `simpsons`, `football`, `politics` | Yes |
| **Era/Platform** | `classic`, `2010s`, `twitter`, `reddit` | Optional |
| **Humor Type** | `surreal`, `wholesome`, `dark-humor`, `absurdist` | Optional |

### Tag Format
- Kebab-case: `homer-simpson` not `homer simpson`
- No spaces within a tag
- Lowercase only
- No special characters except hyphens

### Tag Anti-patterns
| Bad | Good |
|-----|------|
| `homer`, `simpson` (redundant) | `homer-simpson` |
| `funny`, `meme` (too generic) | `reaction`, `template` |
| `a`, `the`, `and` (stop words) | Remove entirely |
| `argentina`, `argentino` | `argentina` only |

## Search Behavior Patterns

Users find memes by searching for:
1. **Character name** + emotion: `"homer angry"`
2. **Scene description**: `"simpson strangling bart"`
3. **Format name**: `"drake hotline bling template"`
4. **Usage context**: `"reaction image happy"`
5. **Language-specific**: `"meme argentina simpson"`

Optimize titles and tags for these search patterns.
