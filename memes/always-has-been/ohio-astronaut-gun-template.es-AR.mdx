# Template del Astronauta con Pistola: Siempre Fue Así

El template del meme "Always Has Been" mostrando dos astronautas en el espacio: uno le pregunta algo al otro, y el segundo le apunta con una pistola mientras dice "siempre fue así". El globo de diálogo vacío lo hace perfecto para adaptarlo a cualquier situación argentina donde una verdad incómoda resulta ser cosa de siempre.
