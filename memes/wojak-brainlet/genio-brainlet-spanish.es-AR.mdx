# El Brainlet Rubio Se Llama a Sí Mismo Genio

Variante del Brainlet Wojak con pelo rubio gritando "GENIO" de forma sarcástica. Para cuando alguien hace algo tan básico que lo celebra como si hubiera descubierto la penicilina, o para ironizar sobre la propia estupidez de uno.
